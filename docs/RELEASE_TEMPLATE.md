## Release artifacts / Артефакты релиза

| File | Format | Purpose |
|------|--------|---------|
| `awg-proxy-{arch}.tar.gz` | OCI | RouterOS 7.x (standard format) |
| `awg-proxy-{arch}-7.20-Docker.tar.gz` | Classic Docker | RouterOS 7.20 LT (legacy format) |
| `awg-proxy-linux-{arch}` | Binary | Run without Docker (Linux) |

> If you don't know which format to choose — use **OCI** (`awg-proxy-{arch}.tar.gz`).
> The **7.20-Docker** format is only needed for RouterOS version 7.20 LT.
